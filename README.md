## Улучшенная производительность

Это более производительная версия конфигурации Neovim, использующая современные линтеры и инструменты для проверки кода:

- **Линтеры:** Biome, Oxlint — обеспечивают быструю и точную проверку качества кода.
- **Линтинг и проверка орфографии:** Встроенная проверка орфографии для английского (`en`) и русского (`ru`) языков.

Это обеспечивает более эффективную работу, быстрый фидбэк и высокий уровень качества разработки.

## Основные команды и карты

| Клавиша | Действие | Описание |
| -------- | -------- | -------- |
| `;` | `:` | Вход в командный режим (эквивалент `:`) |
| `<leader>w` | `:w` | Сохранить текущий файл |
| `jj` | `<ESC>` | Выход из режима вставки |
| `gd` | `vim.lsp.buf.definition()` | Перейти к определению через LSP |
| `grt` | `vim.lsp.buf.type_definition()` | Перейти к типу определения через LSP |

## Окна и перемещение

| Клавиша | Действие | Описание |
| -------- | -------- | -------- |
| `<c-w><leader>` | `which-key` показывает подсказки | Вызов окна управления окнами |
| `<Esc><Esc>` | `vim.cmd.nohlsearch` | Очистить подсветку поиска |
| `k` / `gk` | `gk` | Перемещение вверх по перенесенным строкам |
| `j` / `gj` | `gj` | Перемещение вниз по перенесенным строкам |
| `<C-h>` | `wincmd h` | Влево |
| `<C-j>` | `wincmd j` | Вниз |
| `<C-k>` | `wincmd k` | Вверх |
| `<C-l>` | `wincmd l` | Вправо |

---

## Манипуляции с линиями и окнами

| Клавиша | Действие | Описание |
| -------- | -------- | -------- |
| `>` | `>>` | Увеличить отступ (выравнивание) |
| `<` | `<<` | Уменьшить отступ |
| `<M-j>` | `:m .+1<CR>==` | Переместить текущую строку вниз |
| `<M-k>` | `:m .-2<CR>==` | Переместить текущую строку вверх |
| `<leader>-` | `split` | Горизонтальный сплит |
| `<leader>|` | `vsplit` | Вертикальный сплит |

---

## Миграция строк (перемещение выделенного блока)

| Клавиша | Действие | Режим |
| -------- | -------- | -------- |
| `<M-j>` | `:m '>+1<CR>gv=gv` | Визуальный режим, перемещение вниз |
| `<M-k>` | `:m '<-2<CR>gv=gv` | Визуальный режим, перемещение вверх |

---

## Прочие

| Клавиша | Действие | Описание |
| -------- | -------- | -------- |
| `<leader>-` | `vim.cmd.split` | Создать горизонтальный сплит |
| `<leader>|` | `vim.cmd.vsplit` | Создать вертикальный сплит |
| `<leader>ss` | `setlocal spell!` | Включить/выключить проверку орфографии |
